Servicios

FrontEnd

El desarrollo frontend se enfoca en la interfaz de usuario y la experiencia del cliente, utilizando lenguajes de marcas y javascript para construir la interacción y diseño de una aplicación web o móvil.

Backend

El desarrollo backend se enfoca en la programación y construcción de los servicios y funciones que se ejecutan en el servidor, como la lógica de negocio, el manejo de bases de datos y la comunicación con el frontend.

Diseño Gráfico

El diseño gráfico se enfoca en la creación de elementos visuales y estéticos, utilizando herramientas y programas de diseño como Adobe Photoshop, Illustrator, InDesign y Sketch para crear diseños publicitarios, carteles, ilustraciones, gráficos para web y aplicaciones móviles.
